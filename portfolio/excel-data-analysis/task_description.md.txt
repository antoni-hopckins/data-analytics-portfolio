# Тестовое задание: Анализ данных в Excel

## Общее описание
Задание демонстрирует готовность кандидата к задачам, требующим умения анализировать данные в Excel.

## Задания

### Задание 1
**Дано:** два массива данных с одинаковыми условиями: лист "МК" и "Заявки".
**Задание:** Добавить (с помощью функций Excel) на лист "МК" столбцы с листа "Заявки" - Клиент и Примечание к акту.

### Задание 2
**Дано:** массив с выгрузкой (лист "Детализация")
**Задание:** Определить в скольких заявках и в каком количестве присутствует каждая номенклатура в выбранном массиве данных.

### Задание 3
**Дано:** массив с выгрузка (лист "МК")
**Задание:** Построить сводную таблицу, отображающую количество заявок по дате активации и типу паллет для каждой области комплектации.

### Задание 4
**Дано:** массив с выгрузка (лист "Выгрузка из АХ")
**Задание:** Рассчитать время, затраченное на операции каждым сотрудником в разрезе календарных дней (календарным днем считается дата окончания операции).
